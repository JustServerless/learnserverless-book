What is Serverless?

Serverless is a new framework which makes it easy to build modern, high scalable web applications easily.
It utilizes Amazon web services (AWS) and helps you organize and deploy your resources (e.g. Lambdas, DynamoDB, S3, ...) easily.

Watch the AWS re:Invent talk or read some success stories to get excited.

What you'll get

Free content

The free content is available on our blog and helps you understand the fundamentals you need to use the Serverless framework.

Content and code

Serverless is really new and gets better every day to deliver you the cutting edge technology for your next startup unicorn.
