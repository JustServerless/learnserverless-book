What is Serverless?

Serverless is a new framework which makes it easy to build modern, high scalable web applications easily.
It utilizes Amazon web services (AWS) and helps you organize and deploy your resources (e.g. Lambdas, DynamoDB, S3, ...) easily.

Watch the AWS re:Invent talk or read some success stories to get excited.

What you'll get

Free version

The free version of the book contains a step-by-step process where we'll create a Serverless application from start to finish.

Content and code

Learn and understand every aspect of the Serverless framework while building awesome apps.

Updates

Awesome people contribute to the Serverless framework on a daily basis. We'll keep the book updated to reflect those changes.
